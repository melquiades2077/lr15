# Описание проекта

React-проект с использованием компонентного подхода. Сайт состоит из трёх основных компонентов: Head, Main и Footer. В компоненте Main реализована трёхколоночная структура через CSS Grid в пропорции 1:3:1. Для маршрутизации использовалась библиотека React Router DOM. В зависимости от выбранного пункта меню отображаются страницы Новости, О проекте и Контакты. Для неизвестных маршрутов реализована страница 404. Все требования лабораторной работы выполнены, а интерфейс оформлен с использованием современных CSS-стилей и Grid-разметки.
